# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
赤いボム兵と黒いボム兵を各色の安全地帯に正確に置きスコアを重ねる仕分けゲーム。
/ゲームオーバー条件
安全地帯にないボム兵が一定時間経つと爆発する or 間違えて違う色の安全地帯に置いてしまう。

## ゲームの実装
###共通基本機能
* 背景画像を読み込み・描画

### 担当追加機能
* 安全地帯のデザインと枠のデザインと座標の確認図形を作成。
