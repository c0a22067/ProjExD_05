# ボムへいをわけろ！
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
一定間隔ごとに出現するボムへい！
正しく選別して爆発しないように仕分けよう！

## ゲームの実装
###共通基本機能
* 背景画像の読み込み
### 担当追加機能

* ホーム画面を設ける
* ボムへいの画像を読み込みランダムな動きを行う
* ボムへいが動く範囲を指定する
* 仕分けるスペースを設ける
